Для тех кто совсем не знает что такое Stream Api

Stream API это новый способ работать со структурами данных в функциональном стиле. Чаще всего с помощью stream в Java 8 
работают с коллекциями, но на самом деле этот механизм может использоваться для самых различных данных.
